THE LEVEL EDITOR

ABOUT

This is one of the BIGGEST side projects I’ve worked on. This is a application that I wrote in attempt to learn more about file handling and modification. The general idea is to modify an existing video game. I've also included features such as a grid system, level swapping, statistics and tile data.

DEVELOPER TOOLS

  • Visual Studio
  • C# .NET Windows Forms
  • C# .NET WPF

HEX EDITING

On a very technical level, this is what modifying the file would look like. I can understand what this is saying, but someone else may not. We are looking at one of the levels, in this case. Instead of making modifications to this level through hex, I decided to write an application that would visualize the data instead.

LEVEL SWAPPER

Sometimes, it's easy to develop levels that may feel easier or difficult than other levels. Because of this, I added a neat level swapper within the application!

PALETTE EDITING

The palette editor is designed in its own window. Like level data, I had to find the address for the palette within the file itself, so that I could manipulate the values through my windows form application and save it back into the file. This part of the project aids in making modification to the colors within the game.

DRAWING CANVAS

From here, things get a lot more … technical. I started learning about Node.js and decided to create a Discord bot. This bot will be used within my discord server by any of my users in attempt to obtain information related to their profile. I started off by drawing a canvas. This canvas will hold any user information on it, such as their username, the date in which they joined the server, picture and avatar. I also implemented a fun leveling system!
